# 本地求职助手（爬取 + 本地AI精排 + 自动投递）

> **原创作者**：B站 [@1285452862](https://space.bilibili.com/352671558) ｜ 视频教程与更新都在 B 站，欢迎关注。转载/二次分发请保留署名。

一个面向**城市求职**的本地化求职自动化工具：从招聘网站爬岗位 → 本地大模型按你的背景智能精排 → 一键生成可筛选的网页视图 → 按"专业是否对口"自动投递 → 后台监控 HR 回信。**全程本地运行、数据不出本机**，AI 用本机 Ollama，不需要任何付费 API。

> ⚠️ **免责声明**：本项目仅供学习与个人求职效率研究。自动投递使用你**本人**的招聘平台账号，请遵守相应平台的用户协议、合理控制频率，由此产生的账号风险由使用者自行承担。请勿用于骚扰或大规模滥用。

## ✨ 主要特性
- **个性化引擎**：填一次"我的资料"（城市/学历/背景/想做的方向），系统用本地大模型自动算出该搜的岗位关键词、该跳过的不对口岗位、招呼语——**会计/设计/运营/IT 等不同背景都适用**。
- **本地 AI 精排**：用 Ollama + Qwen3 对岗位做语义打分（强烈推荐/可投/观望/不建议），过滤高薪诈骗、僵尸号、重体力岗。
- **四层投递过滤**：评级不建议 / 风险词（带"劝告语气"识别，不误杀）/ 岗位名黑名单（按你的背景生成）/ **投递时读真实职位描述做 AI 对口核查**（能抓出"会CAD/持证/现场设备"这类藏在描述里的硬要求）。
- **自动投递 + 保号**：每日上限、随机间隔、去重、被平台限额自动停。
- **回信监控**：区分"你投的岗位的回信"和"广告骚扰"，标出新回信。
- **可筛选网页视图** + **桌面 GUI**（一键全自动 / 分步皆可）。
- **一键安装**：在线版自动下载依赖与模型；离线版可把模型一并打包（解决国内下载难）。

## 🚀 安装与使用
> 需要 Windows + Chrome 浏览器。AI 精排需要本机 [Ollama](https://ollama.com)（安装器会自动装）。

**方式一：在线安装（拉本仓库代码）**
1. 下载本仓库（绿色 `Code` → `Download ZIP`，或 `git clone`）。
2. 双击 **`一键安装.bat`**：自动装 Python、依赖、Ollama，并拉取 AI 模型 `qwen3:8b`。
   - 🇨🇳 国内拉模型可能慢/失败，建议挂代理，或用下面的"离线包"。
3. 双击桌面【求职助手】→ 点【🧑 我的资料】填情况 → 点【🌐 打开网站·登录BOSS】扫码登录 → 开用。

**方式二：离线整包（推荐给非技术用户）**
把模型/Ollama/Python 一并打好的 ~6GB 离线包，对方双击 `一键安装.bat` 即可、无需任何下载。
- 自己制作：开着代理运行 `python 制作离线包.py`，产物在 `D:\求职助手离线包`，整个文件夹用网盘/U盘分发。
- 离线包下载（夸克网盘）：https://pan.quark.cn/s/97cc55746ee7 　提取码：**YCLQ**

## 🧩 技术栈
Python · [DrissionPage](https://github.com/g1879/DrissionPage)（接管真实 Chrome 抓取/投递）· Ollama + Qwen3（本地 LLM 精排与对口核查）· Tkinter GUI · openpyxl/pandas。

## 📂 主要模块
| 文件 | 作用 |
|---|---|
| `job_assistant_app.py` | 桌面 GUI 主程序 |
| `boss_scraper.py` | 岗位爬取（DrissionPage 接管已登录 Chrome） |
| `job_matcher.py` / `local_ai_matcher.py` / `merge_analysis.py` | 规则筛选 + 本地 AI 精排 + 出推荐表 |
| `profile_engine.py` | 个性化引擎（按背景生成关键词/对口规则/招呼语） |
| `auto_apply.py` | 自动投递（四层过滤 + 保号 + 投前AI对口核查） |
| `reply_monitor.py` | 回信监控 |
| `build_web.py` | 推荐表 → 可筛选网页 |
| `auto_run_all.py` | 一条龙全自动编排 |
| `install.py` / `制作离线包.py` | 一键安装 / 离线打包 |
